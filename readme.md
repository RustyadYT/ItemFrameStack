## What is ItemFrameStack?
ItemFrameStack lets you store up to 4 items in a single item frame.
Frames with multiple items cycle through them every 2 seconds —
perfect for shop displays, sorted storage walls, or decoration.

📦 [Download on Hangar](https://hangar.papermc.io/RustyLabs/ItemFrameStack)

## How to Use

| Action | Result |
|---|---|
| Right-click frame with item | Add item (max 4) |
| Right-click frame with empty hand | Rotate current item |
| Sneak + Right-click frame | Remove current item |
| Sneak + Left-click frame | Rotate current item |
| Left-click empty frame | Break frame (drops it) |

## Commands

| Command | Description |
|---|---|
| /ifs help | Show usage |
| /ifs info | Show version and author |

## Permissions

| Permission | Default | Description |
|---|---|---|
| itemframestack.use | true | Use item frame stacking |
